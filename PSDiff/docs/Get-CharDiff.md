# Get-CharDiff

## SYNOPSIS
Get a diff of two strings character by character

## SYNTAX

```
Get-CharDiff [-Text1] <Object> [-Text2] <Object>
```

## DESCRIPTION
Function from https://github.com/google/diff-match-patch

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-CharDiff 'this and that' 'tit and tat'
```
